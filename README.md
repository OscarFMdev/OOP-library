## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need [PostgreSQL](https://www.postgresql.org/) installed, [git](https://git-scm.com/downloads), a text editor like [Visual Studio Code](https://code.visualstudio.com/download), and [Ruby](https://www.ruby-lang.org/en/downloads/)

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone git@github.com:OscarFMdev/OOP-library.git
```


### Install

Install this project with:

```sh
  cd my-project
  gem install
```
